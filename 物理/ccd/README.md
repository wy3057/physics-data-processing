# CCD 干涉实验数据处理脚本

本目录包含两个独立的小脚本，用于课堂 CCD 干涉/衍射实验中两张原始表格的快速计算。

## 文件说明

| 文件 | 功能 |
| --- | --- |
| `ccd数据表一处理.py` | 通过输入光路距离和 CCD 上 -1、+1 级光强点的位置，计算单缝宽度以及与理论值的百分误差。脚本内提供了毫米/微米与米之间的转换函数，避免手工换算。 |
| `ccd表二.py` | 根据实验表二里测得的 $v_{0\min}$、$v_{-1}$、$v_{+1}$ 计算 $v_0$，便于完成剩余推导。 |

## 使用方法

1. 在终端进入当前目录，按需运行脚本：
   ```bash
   python ccd数据表一处理.py
   # 或
   python ccd表二.py
   ```
2. 根据提示依次输入实验记录（距离、像素坐标或电压读数）。
3. 程序会在终端打印计算出的单缝宽度、百分误差或 $v_0$。

## 常见问题

- `ccd数据表一处理.py` 默认波长 650 nm、像素间距 11 μm，如需调整可直接修改脚本顶部常量。
- 两个脚本都依赖于标准 Python，无需额外第三方库。
