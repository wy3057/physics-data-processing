# 磁致伸缩实验处理工具

该目录提供 2 个配套脚本，帮助完成磁致伸缩实验的数据整理、特性曲线绘制。

## 文件结构

| 文件 | 功能 |
| --- | --- |
| `磁致伸缩处理完整版.py` | 从命令行批量输入电流 $I$ 与干涉条纹计数 $n$，自动计算螺线管内磁场强度 $H$、长度变化 $\Delta L_1$、相对伸长 $\Delta L_1 / L_1$，并绘制 $\Delta L_1 / L_1$ 对 $H$ 的特性曲线。 |
| `绘图程序.py` | 已经拥有 $H$ 与 $\Delta L_1 / L_1$ 数据时，单独绘制折线图，方便重复画图或复核报告。 |

## 运行方式

```bash
# 完整计算流程
python 磁致伸缩处理完整版.py

# 仅根据现成数据绘图
python 绘图程序.py
```

运行后按照提示逐行输入 `数值,数值` 形式的数据，输入 `done` 结束。程序会打印计算结果并在窗口中弹出图像。

## 参数约定

- 默认匝数 `N=1200`、线圈长度 `L=0.07 m`、原始长度 `L1=0.11 m`、激光波长 632.8 nm。如与实验不同，可直接修改脚本顶部常量。
- 绘图脚本默认将 $H$ 转换到 $10^3$ A/m、$\Delta L_1 / L_1$ 转换到 $10^{-6}$ 量级，符合实验报告图例。
