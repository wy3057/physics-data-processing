## 项目概览

- **目标**：让常见实验在“导入 CSV → 生成结果/图表/报告”的流程中一键完成，减少重复手算。
- **输入/输出**：大多数脚本接收表格（CSV/Excel）或图片，导出处理后的 Excel、PNG 及报告模板。
- **环境**：推荐 Python ≥ 3.9，并在各实验目录下提供了示例数据与 README，帮助快速定位使用方法。

## 化工原理流程分析

### 伯努利方程验证（流体能量转换）

- 计算各截面流速、静压头、动能头、总水头。
- 绘制测量点高度示意图（柱状图）与能量转换位置-压强图（散点图）。
- 支持 6 组数据批量分析，并生成完整实验报告。
- 基于伯努利方程 `p/γ + z + α*v²/(2g) = 常数`，可分析压头损失与流量关系。

### 流动过程实验

- 研究层流/湍流切换，直观呈现压力分布与雷诺数-流动形态曲线。
- 计算管道阻力系数，输出报告模板，便于直接撰写实验结论。

### 干燥、精馏等模块

- 各目录均提供 README 说明如何导入实验数据、配置物性参数并生成图表。

## 生物化学自动化脚本

### 植物可溶性蛋白（考马斯亮蓝法）

- 一键拟合标准曲线，输出斜率、截距与 R²。
- 计算比色体系与原提取液中的蛋白浓度，生成 `protein_results.xlsx` 和标准曲线图。
- 支持多样品批量分析，并在终端打印可直接粘贴到实验报告的摘要。
