# 固定床干燥实验数据处理

`main.py` 实现了教材中固定床干燥实验的整套数据处理：读取原始表格、计算含水量/平均含水量/干燥速率、自动识别恒速段、求传热系数、孔板流量和空气流速，并生成曲线与结果表。

## 使用步骤

1. 将实验记录整理为 CSV/Excel，列名需与脚本顶部常量一致（默认：`time_min`、`GT_g`、`湿球温度摄氏度`、`压差（kpa）`、`干球温度`、`入口温度`）。
2. 运行程序：
   ```bash
   cd 化工原理/干燥
   python main.py
   ```
3. 按提示输入数据文件路径以及实验常数（支撑架质量、绝干物料量、干燥面积等）。
4. 处理完成后会输出：
   - `干燥数据_处理结果.xlsx`
   - `干燥曲线_X_t.png`
   - `干燥速率曲线_U_Xav.png`

## 重要参数

- 孔板系数 `C0`、孔径 `d0`、通道截面积 `A_tunnel` 以及湿球潜热 `r_tw` 在脚本前部配置，可根据装置情况调整。
- 程序会尝试自动识别恒速段，如出现 NaN，可检查数据中是否存在重复时间或不足的有效点。
