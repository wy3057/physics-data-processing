# 伯努利方程验证实验工具

本目录提供伯努利实验（流体能量转换实验）的全流程数据处理脚本，涵盖数据导入、参数计算、可视化与报告生成。

## 目录结构

| 文件 | 作用 |
| --- | --- |
| `main.py` | 主程序。读取 6 组测量点压强数据，计算各截面静压头/动能头/总水头、压头损失，并自动绘制图三～图六以及生成文字报告。 |
| `csv生成.py` | 生成示例 `experiment_data_combined.csv`，便于快速体验主程序。 |
| `experiment_data_combined.csv` | 示例数据，包含 6 组流量和 15 个测压点读数。 |

## 快速开始

1. 准备与示例格式相同的 CSV 文件：第一列为分组标签，第二列为流量 (l/h)，后 15 列为测压点读数 (mmH₂O)。
2. 运行：
   ```bash
   cd 化工原理/伯努利
   python main.py
   ```
3. 根据提示输入 CSV 路径（可以直接使用 `experiment_data_combined.csv`）。
4. 程序会依次在终端输出各截面计算结果、压头损失，并弹出 4 张图，最后打印实验报告摘要。

## 自定义

- 若实验设备的管径、测点位置不同，可在 `main.py` 顶部修改直径、高度等常量。
- 图表字体目前使用 SimHei，如运行环境没有该字体，可替换为其他中文字体或注释相关代码。
