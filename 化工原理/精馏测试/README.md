# 乙醇-正丙醇精馏塔计算程序

该目录内的 `main.py` 复现了精馏实验中 McCabe-Thiele 图解的计算过程，支持全回流和有限回流两种模式，可输出理论板数、全塔效率以及 Murphree 板效率。

## 功能概览

- 通过折光指数转换质量分数，再换算为摩尔分数。
- 根据平衡表（t-x-y 数据）插值获取气液平衡关系、泡点温度。
- 计算进料热状况参数 `q`，构建 q 线、精馏段/提馏段操作线。
- 根据输入的产品、釜液组成绘制 McCabe-Thiele 阶梯并统计理论板数。
- 输出结果 DataFrame 并支持保存 CSV，同时提供示意图。

## 使用方式

```bash
cd 化工原理/精馏测试
python main.py
```

脚本会提示输入：产品与釜液折光指数/浓度、进料流量和热状况、实际塔板数、操作回流比等。根据输入选择运行全回流或有限回流计算分支。

如需更新物性数据，可修改文件顶部的 `T_data`、`x_data`、`y_data` 或比热/潜热参数。
