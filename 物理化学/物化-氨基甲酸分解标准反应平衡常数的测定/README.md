# 氨基甲酸铵分解平衡常数计算

`main.py` 依据教材第 2-4 节示例，处理 NH₄COONH₄(s) ⇌ 2NH₃(g) + CO₂(g) 反应的实验数据，计算平衡常数及热力学参数。

## 主要功能

- 支持手动输入多组温度与平衡压力，或直接使用脚本内置的参考数据表。
- 根据平衡压力计算 $K^\circ = \frac{4}{27}(p/p^\circ)^3$ 并输出对数值。
- 详细展示 ΔH°、ΔG°、ΔS° 的逐步推导，并比较由热力学数据和实验拟合得到的结果。
- 通过 Arrhenius 线性拟合 `lnK` vs `1/T` 得到反应焓变、熵变等，最后可将结果写入文本文件。

## 运行方法

```bash
cd 物理化学/物化-氨基甲酸分解标准反应平衡常数的测定
python main.py
```

根据提示选择“手动输入”或“使用参考数据”，之后跟随交互完成所有数据录入。程序输出的每个中间值都带公式说明，可直接引用到实验报告。
