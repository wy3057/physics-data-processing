# 蔗糖水解速率常数测定

本目录内 `main.py` 支持 1 mol·L⁻¹ 与 2 mol·L⁻¹ HCl 两种催化剂浓度的数据录入，自动计算蔗糖水解的速率常数并生成表格/CSV。

## 程序流程

1. 收集实验基本信息（实验者、日期、温度、旋光管长度等）。
2. 读取 `α_0`、`α_∞`、零点校正等初始参数。
3. 交互式输入时间-旋光度数据（支持两种酸浓度同时比较）。
4. 计算 `α_t-α_∞`、`\ln(α_t-α_∞)`、反应进度、旋光度变化百分比等列。
5. 通过线性回归得到速率常数 `k`、半衰期，并输出格式化表格（终端 + CSV）。

## 使用说明

```bash
cd 物理化学/物化-蔗糖水解测定
python main.py
```

运行后按照提示逐条输入 `时间,旋光度`，输入 `done` 结束。程序会生成 `sucrose_hydrolysis_data_*.csv`，可直接附在实验报告中。
