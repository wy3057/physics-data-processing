# 乙醇饱和蒸气压与蒸发焓计算

`delath.py` 将克劳修斯-克拉贝龙方程的计算流程脚本化，提供交互式数据录入、详细的逐步推导以及线性回归拟合。

## 功能特性

- 引导式输入样品名称、室温、大气压和多组温度/压差数据。
- 自动计算 $T$、$1/T\times 10^3$、饱和蒸气压 $p= P_{atm}-\Delta p$ 及 $\lg p$，并逐点输出计算细节。
- 使用最小二乘拟合 `lgp` 与 `1/T` 关系，得到斜率/截距和决定系数 $R^2$。
- 根据斜率求取蒸发焓，输出预测值与残差表，便于直接整理到报告中。

## 运行方式

```bash
cd 物理化学/物化-乙醇饱和蒸汽压的计算
python delath.py
```

脚本依赖 `pandas`、`numpy`、`scipy` 和 `matplotlib`。若缺少中文字体，程序会自动fallback，不影响计算。
