# 乙酸乙酯皂化反应速率常数计算

`main.py` 用于处理乙酸乙酯在不同温度下的电导率随时间数据，自动计算速率常数、拟合直线并求活化能，附带详细的终端输出，方便整理实验报告。

## 核心能力

- 将 `1/(L_0-L_t)` 与 `1/t` 做线性回归，得到速率常数 `k`、斜率 `m`、截距 `B` 及相关系数。
- 自动打印每个数据点的计算表，包括 `t`、`L_t`、`L_0-L_t`、`1/(L_0-L_t)`、`1/t`。
- 根据多温度下的 `k` 值使用 Arrhenius 方程计算活化能，输出 J/mol 与 kcal/mol。
- 生成图像和 Excel/文本表格（详见脚本内的 Pandas 输出逻辑）。

## 使用方式

```bash
cd 物理化学/物化-乙酸乙酯皂化的测定
python main.py
```

根据交互式提示输入不同温度下的时间、电导率数据及起始电导率 `L0`。脚本会直接给出速率常数和活化能计算过程，可将终端输出复制进实验报告。
